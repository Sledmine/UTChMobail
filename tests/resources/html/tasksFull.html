III.2. Desarrollo de base de datos con transacciones (30%) está en fecha de entrega

domingo, 5 julio, 21:12

Desarrollar un sistema de base de datos siguiendo las siguientes instrucciones:

  1. Crear base de datos llamada 'central', para el control de ocupación de vehículos de transporte foráneo.
    1. crear en la base de datos 'central' las siguientes tablas, los campos son creados según diseño propio
    2. transporte(almacena información de los vehículos de transporte. placa, marca, etc.)
    3. destino(almacena información de los destinos: nombre, distancia en kilómetros, etc.)
    4. boleto(almacena información de los boletos vendidos, número, hora salida, llegada, costo, etc.)
  2. Crear una base de datos llamada terminal, idéntica a la base de datos 'central'
  3. Enviar información a las tablas de ambas bases de datos controlando transacciones
  4. Las bases de datos pueden estar en el mismo servidor.

Enviar:

Archivo de restauración de la base de datos.

Documento digital con el diseño de las bases de datos, explicación del funcionamiento de los procedimientos almacenados y transacciones y conclusiones.
